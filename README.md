# Boostings
В репозитории содержатся работы, посвященные ансамблевым методам машинного обучения с акцентом на градиентный бустинг. В файлах представлены примеры использования популярных библиотек, подбор гиперпараметров и решение задач регрессии и классификации.

### 1. 11_3_Gradient Boosting.ipynb

**Введение в градиентный бустинг (Scikit-learn)**

* Демонстрация работы `GradientBoostingRegressor` на классическом датасете California Housing.
* Сравнение производительности случайного леса (Random Forest) и бустинга.
* Визуализация процесса переобучения бустинга при увеличении количества деревьев.
* Базовая стратегия настройки `learning_rate` и `n_estimators`.

### 2. 12_2_Catboost.ipynb

**CatBoost: Продвинутая работа с категориальными признаками**

* Решение задачи классификации на датасете Titanic.
* Особенности библиотеки от Яндекса: автоматическая обработка категориальных данных и пропусков.
* Использование встроенной визуализации обучения и кросс-валидации.
* Анализ важности признаков (Feature Importance).

### 3. 12_3_LightGBM.ipynb

**Сравнительный анализ: XGBoost, CatBoost и LightGBM**

* Практическое сравнение трех "китов" градиентного бустинга на одной задаче.
* Использование `GridSearchCV` для оптимизации параметров.
* Замеры времени обучения и качества моделей (метрика R^2).

### 4. 12_4_Optuna.ipynb

**Автоматическая оптимизация гиперпараметров с Optuna**

* Основы работы с фреймворком Optuna.
* Интеграция Optuna с LightGBM и Scikit-learn пайплайнами.
* Демонстрация поиска оптимальных параметров в широком диапазоне с использованием логарифмического шага для learning_rate.

### 5. 12_5_Дз Бустинги.ipynb

**Практическая работа: Прогнозирование задержек авиарейсов**

* Полноценный кейс по прогнозированию задержек самолетов (бинарная классификация).
* Предобработка реальных данных, работа с типами данных category для LightGBM.
* Применение Optuna для максимизации метрики ROC-AUC.
* Сравнение моделей "из коробки" и тюнингованных алгоритмов.
